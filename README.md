# rollup-library-ts
rollup+Ts快速开发库、插件模板


### 安装
```
pnpm i
```

### 开发
```
pnpm run dev
```

### 打包
```
pnpm run build
```
  
### 本地安装
```
npm link
```

### 测试命令
```
pnpm run test
```

### 这个命令会列出所有被链接的全局 Node.js 包
```
npm ls- g --link
```

npm unlink -g <package-name>
